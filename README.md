# 📖 성인 교과 (Bible Study Quiz App)

안식일학교 장년 교과의 내용을 복습할 수 있는 웹 기반 퀴즈 애플리케이션입니다.  
**NotebookLM**의 깔끔하고 직관적인 UI 디자인을 벤치마킹하여 학습 몰입도를 높였습니다.

![Project Status](https://img.shields.io/badge/Status-Active-brightgreen)
![Tech Stack](https://img.shields.io/badge/HTML5-CSS3-JavaScript-blue)

## ✨ 주요 기능 (Features)

* **NotebookLM 스타일 UI**: 텍스트 중심의 깔끔하고 모던한 카드형 디자인을 적용했습니다.
* **반응형 웹 디자인 (Responsive)**: PC와 모바일(스마트폰, 태블릿) 어디서든 최적화된 화면을 제공합니다.
* **즉각적인 피드백**: 정답/오답 여부와 함께 교과 내용을 바탕으로 한 상세한 해설을 즉시 확인할 수 있습니다.
* **점수 및 결과**: 문제를 모두 푼 후 최종 점수와 격려 메시지를 제공합니다.
* **Single File**: HTML, CSS, JavaScript가 하나의 파일(`index.html`)에 통합되어 있어 배포와 관리가 매우 간편합니다.

## 🛠️ 기술 스택 (Tech Stack)

* **HTML5**: 시맨틱 마크업 구조
* **CSS3**: Flexbox 레이아웃, CSS Variables를 활용한 테마 관리, 반응형 미디어 쿼리
* **JavaScript (Vanilla)**: 외부 라이브러리 없이 순수 JS로 구현된 퀴즈 로직

## 🚀 실행 방법 (How to Run)

### 로컬에서 실행하기
1. 이 저장소를 클론(Clone)하거나 ZIP 파일로 다운로드합니다.
2. `index.html` 파일을 더블 클릭하여 웹 브라우저(Chrome, Safari, Edge 등)에서 엽니다.

### GitHub Pages로 배포하기 (온라인 실행)
이 프로젝트는 **GitHub Pages**를 통해 무료로 호스팅할 수 있습니다.
1. 저장소의 **Settings** 탭으로 이동합니다.
2. 좌측 메뉴에서 **Pages**를 클릭합니다.
3. **Branch** 설정에서 `main` (또는 `master`) 브랜치를 선택하고 `/root` 폴더를 지정한 뒤 Save를 누릅니다.
4. 잠시 후 상단에 생성된 주소(URL)를 통해 어디서든 퀴즈에 접속할 수 있습니다.
